# Python Flask Application

## Overview
A simple Python Flask web application template.

## Project Structure
- `src/main.py` - Main Flask application entry point
- `pyproject.toml` - Python package configuration and dependencies
- `static/` - Directory for static assets (CSS, JS, Images)
- `templates/` - Directory for HTML templates

## Running the Application
The application runs on port 5000 using Flask's built-in development server. Run it via the "Flask Server house-keeping" workflow or by running `python src/main.py`.

## Dependencies
- Flask - Web framework
